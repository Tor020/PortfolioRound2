About

At time of writing in 2017, I'm a 26 year old self taught Web Developer whose been using the Internet since the late 90s. I've always thought the Internet was pretty awesome, and figuring out how things work has always been something I've enjoyed doing.

I find Web Development to be really engaging and fun for me personally because it feels like there's great practical application to all the new things I learn. For instance, I loved learning how to make the Pong demo because when I started with Web Development, I had no idea that making games via HTML5 and Javascript was even possible. It was really exciting for me to see that I could utilize the canvas and Javascript to make something like that. It was just really cool for me.

As far as short term goals, I'm really dedicated to understanding Javascript and working to be able to utilize it to its full potential. This includes things like learning the nuances of modern Javascript frameworks. I have long term aspirations of becoming a fullstack developer.
